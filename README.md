# Cannes Lions 2026 - Sito Web

## Struttura del Sito

### Homepage (`index.html`)
- **Hero section** con titolo e statistiche del festival
- **Grand Prix Winners** - 8 card con tutti i vincitori Grand Prix
- **Categorie** - Griglia navigabile per categorie
- **Tutti i Vincitori** - Lista filtrabile (Oro/Argento/Bronzo)
- **Agency of the Year** - Le agenzie premiate

### Case Study Pages
Ogni Grand Prix ha una pagina dedicata con:
- Panoramica della campagna
- La sfida
- La soluzione
- I risultati
- Key learnings
- Sidebar con dettagli e condivisioni

### Design System (Apple-inspired)

#### Colori
- Background: `#FFFFFF` / `#F5F5F7`
- Text: `#1D1D1F` / `#86868B`
- Accent: `#0066CC`
- Gold: `#FFD700`, Silver: `#C0C0C0`, Bronze: `#CD7F32`

#### Typography
- Font: Inter (Google Fonts)
- Sizes: 12px → 64px (xs → 4xl)

#### Componenti
- Navigation fissa con blur effect
- Card con hover animations
- Filtri per medaglie
- Sidebar sticky nelle case study

## File
- `index.html` - Homepage
- `style.css` - Tutti gli stili (Apple design system)
- `script.js` - Filtri, smooth scroll, animazioni
- `case-*.html` - Pagine case study
